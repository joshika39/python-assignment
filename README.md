# Python assignment for University

The assignment is can be accessed [here](https://py-assignment.kou-gen.net/).

## Usage

### Linux

```bash
python -m venv venv

. venv/bin/activate

pip install -r requirements.txt --upgrade

python main.py
```

### Windows

```ps1
py -m venv venv

.\venv\Scripts\Activate.ps1

pip install -r requirements.txt --upgrade

py main.py
```
